function optimizeResourceAllocation(tasks, workers) {
  /**
   * Implementation for the technical challenge
   */
  const result = [];
  const sortedTasks = tasks.sort((a, b) => b.priority - a.priority);

  sortedTasks.forEach(task => {
    const availableWorker = workers.find(w => w.capacity >= task.load);
    if (availableWorker) {
      result.push({ taskId: task.id, workerId: availableWorker.id });
      availableWorker.capacity -= task.load;
    }
  });

  return result;
}

Resource Allocation Optimization

You are tasked with developing a system that assigns a set of incoming tasks to a pool of available workers based on their current capacity and the priority of the tasks.

Requirements

  • Highest priority tasks must be allocated first.
  • A worker can only accept a task if their capacity >= task.load.
  • If multiple workers can handle a task, assign it to the one with the highest current capacity to maintain load balancing.
  • The function must return an array of mappings { taskId, workerId }.

Constraints

  • - tasks: 1 ≤ n ≤ 10,000
  • - workers: 1 ≤ m ≤ 5,000
  • - load/capacity: 1 ≤ x ≤ 100

Example

Input: 
tasks = [{id: 1, load: 20, priority: 5}]
workers = [{id: 'A', capacity: 50}]

Output: 
[{taskId: 1, workerId: 'A'}]
